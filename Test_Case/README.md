# 📊 Test Case 설계 및 검증 범위

본 문서는 남마법사 업데이트의 품질을 보증하기 위한 상세 테스트 범위와 핵심 시나리오를 정의합니다.

---

## 🎯 1. 테스트 범위 (Test Scope)

엑셀 마스터 TC에 포함된 주요 검증 항목입니다.

### 1.1. 메인 TC (핵심 기능 및 크로스 플랫폼)
* **스킬 메커니즘 검증:** PC의 복잡한 커맨드 입력 대비 모바일 '원터치 콤보'의 발동 속도 및 판정 비교.
* **피격/상태이상 판정:** 남마법사 특유의 '불사' 스킬이 던전에서 정상 작동하는지 확인.
* **UX/UI 환경별 HUD 최적화:** PC(FHD/QHD)와 모바일(18:9 등) 해상도 전환 시 쿨타임 가독성 및 터치 영역 확인.
* **성장 이벤트 보상 수령:** 점핑 이벤트 보상(강화권, 아바타 등)의 인벤토리 지급 및 실적용 여부.
* **네트워크 입력 지연(Latency):** PC 버전의 키보드 입력과 모바일 터치 입력 간의 반응 속도(Input Lag) 체감 차이 측정.

### 1.2. 서브 TC (환경 및 호환성)
* **입력 외부 컨트롤러 대응:** 모바일 기기에 게임패드(PS/Xbox) 연결 시 남마법사 전용 스킬 매핑 정상 여부.
* **환경 백그라운드 전환:** 플레이 중 전화 수신, 타 앱 전환 후 복귀 시 '남마법사' 리소스 로딩 오류 확인.
* **성능 발열 및 배터리:** 30분 이상 연속 사냥 시 남마법사의 화려한 이펙트가 프레임(FPS)에 미치는 영향 분석.
* **네트워크 핸드오버:** Wi-Fi에서 LTE/5G로 전환 시 남마법사의 스킬 위치 동기화 오류(고스트 현상) 확인.
