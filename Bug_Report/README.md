# 🐞 결함 보고서 관리 (Bug Reports)

이 디렉토리는 **DNF-M: 남마법사 - 빙결사** 업데이트 시점 당시 발견되었던 주요 결함들을 Jira 포맷으로 기록한 공간입니다. 

> **[공지]** 본 보고서에 포함된 결함들은 현재 **수정 완료(Resolved)**된 상태입니다. 실무 프로젝트 경험을 재현하기 위해 과거 발생했던 이슈를 기획/기술적으로 분석하였으며, 직접 테스트가 불가능한 환경의 경우 타 플레이어의 에비던스(스크린샷)를 인용하여 QA의 시각으로 재구성 및 분석하였습니다.

## 📋 결함 관리 기준 (Issue Classification)

### 1. 심각도 (Severity)
* **Critical:** 앱 강제 종료(Crash), 진행 불가, 데이터 유실 등 치명적인 결함.
* **Major:** 주요 기능 오작동, 연출 누락 등 사용자 경험에 큰 영향을 주는 결함.
* **Minor:** 텍스트 오타, 미세한 그래픽 깨짐 등 경미한 결함.

### 2. 우선순위 (Priority)
* **P1:** 핵심 기능 이슈로 즉시 수정 필요.
* **P2:** 주요 기능 결함으로 수정 권장.
* **P3:** 일정에 따라 추후 수정 가능.
