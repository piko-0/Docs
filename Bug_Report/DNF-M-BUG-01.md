# [Bug] 특정 네트워크 지연 환경에서 '불사' 스킬 미발동 이슈

### 1. 결함 정보
- **Issue ID:** DNF-M-BUG-01
- **Severity:** Critical (치명적)
- **Priority:** High (긴급)
- **Component:** 전투 로직 / 남마법사 공통
- **Environment:** Android 14 (S24), Wi-Fi (Latency 주입 환경)

### 2. 결함 요약
남마법사 캐릭터가 사망에 이르는 치명타를 입었을 때, 특정 네트워크 지연(Ping 150ms 이상) 상황에서 '불사' 로직이 트리거되지 않고 즉시 유령 상태로 진입함.

### 3. 재현 단계 (Steps to Reproduce)
1. 던전(빌마르크 제국 실험장) 진입
2. 캐릭터의 HP를 5% 이하로 조절
3. 네트워크 핸드오버(Wi-Fi <-> LTE) 발생 시점에 보스 몬스터에게 피격
4. '불사' 스킬의 발동 여부 확인

### 4. 실제 결과 (Actual Result)
- '불사' 스킬의 쿨타임이 없는 상태임에도 불구하고 로직이 생략되며 즉시 사망 처리됨.

### 5. 기대 결과 (Expected Result)
- 어떤 네트워크 환경에서도 HP가 0이 되는 순간 '불사' 무적 로직이 최우선으로 가동되어야 함.
