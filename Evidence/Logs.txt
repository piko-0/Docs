# 📑 Test Log: Glacial Master Defect Analysis

이 로그는 빙결사 캐릭터의 핵심 결함 2종에 대한 데이터 분석 및 검증 기록입니다.

---

### [TEST LOG: BUG-001 - Armor Mastery Data Inversion]

※ Note: 본 로그는 결함 발생 당시 타 플레이어의 검증 자료를 인용하여 QA 관점에서 재분석한 기록입니다.

-----------------------------------------------------------
**1. 장비 미착용 상태 (Base Line)**
- 캐릭터 기본 HP: 13,459
-----------------------------------------------------------

**2. 15레벨 가죽 상의 착용 시 (Lv.15 Leather Top)**
- 표기 HP: 14,018 (전체 증가량: +559)
- 세부 데이터 분석:
  * 상의 아이템 고유 HP 옵션: +256
  * 빙결사 가죽 마스터리 실적용 수치 이상
  * 결과: 기본 HP(13,459) + 상의(256) + 마스터리 가산 = 14,018
-----------------------------------------------------------

**3. 25레벨 가죽 상의 착용 시 (Lv.25 Leather Top) - [결함 발견]**
- 표기 HP: 13,914 (전체 증가량: +455)
- 세부 데이터 분석:
  * 상의 아이템 고유 HP 옵션: 없음 (0)
  * 빙결사 가죽 마스터리 기대값: +2,851 (툴팁 기준 계수)
  * 결과: 13,459 + 2,851 = 16,310이 되어야 하나, 실제로는 13,914 기록.
-----------------------------------------------------------

**[QA 분석 결론]**
- **데이터 역전 현상:** 15Lv 상의 착용 시(14,018)보다 상위 등급인 25Lv 상의 착용 시(13,914) 최종 HP가 감소함.
- **로직 오류 증명:** 방어구 마스터리의 기대 수치가 연산에서 완전히 누락되어 기본 스탯만 반영되는 결함 확인.
-----------------------------------------------------------

### [TEST LOG: BUG-002 - Hailstorm Hit Detection Failure]

※ Note: 본 로그는 결함 발생 당시 유저 제보 데이터를 기반으로 패치 전/후 수련장 리포트를 QA 관점에서 비교 분석한 기록입니다.

-----------------------------------------------------------
**1. 패치 전: 결함 상태 (Pre-Patch: Issue Identified)**
- **평균 데미지:** **185,148**
- **세부 데이터 분석:**
  * 전체 스킬 중 데미지 점유율 **10위** 기록 (하위권).
  * 현상: 시전 시 생성되는 추가 얼음 창의 타격 판정이 간헐적으로 증발하는 로직 오류 발생.
-----------------------------------------------------------

**2. 패치 후: 정상화 검증 (Post-Patch: Verification)**
- **평균 데미지:** **473,641,771**
- **세부 데이터 분석:**
  * 전체 스킬 중 데미지 점유율 **3위** 기록 (상위권 진입).
  * 패치 전(18.5만) 대비 평균 데미지 약 **2,500배 이상** 상승 확인.
  * 결과: 누락되었던 추가 타격 판정이 정상 복구됨에 따라 스킬 기대 데미지 산출 성공.
-----------------------------------------------------------

**[QA 분석 결론]**
- **결함 증명:** 패치 전/후 데이터의 극명한 차이(185,148 vs 473,641,771)를 통해, 이전 로직에서 심각한 타격 판정 누락이 존재했음을 수치적으로 증명함.
- **최종 판정:** 1/29 업데이트를 통해 핵심 딜링 메커니즘이 정상화되었으며, 재검증 결과 주력 스킬로서의 기능 수행에 결함이 없음을 확인 (Resolved).
-----------------------------------------------------------
