# [QA] Game QA Portfolio

본 문서는 **던전앤파이터 모바일 v1.22 업데이트**를 통해 출시된 신규 클래스 '남마법사'의 안정적인 라이브 서비스를 위한 테스트 계획을 기술합니다.

---

## 1. 테스트 개요 및 목적 (Test Summary)
신규 클래스 '남마법사'의 전투 메커니즘 무결성 및 플랫폼별(Mobile/PC) 상호 운용성 검증을 목적으로 합니다. 라이브 서버 빌드 안정성 및 플랫폼 간 형평성 검증을 위해 **Black-box, Compatibility, Regression Testing** 전략을 채택하며, 사용자 경험(UX) 저해 요소를 사전에 식별하고 조치하는 데 중점을 둡니다.

* **기능 및 성능 검증:** 신규 클래스 '남마법사'의 기능적 결함 및 성능 저하 확인.
* **플랫폼 최적화 분석:** PC 클라이언트와 모바일 기기 간의 조작감 및 UI 최적화 차이 분석.
* **네트워크 신뢰성 검증:** 실제 네트워크 환경(LTE/5G/Wi-Fi)에서의 세션 유지 및 동기화 검증.

---

## 2. 테스트 범위 (Test Scope)

### 2.1 메인 TC (60~70%): 핵심 기능 및 크로스 플랫폼 비교
| 구분 | 테스트 항목 | 상세 내용 (라이브 서버 기준) |
| :--- | :--- | :--- |
| **전투** | 스킬 메커니즘 검증 | PC의 복잡한 커맨드 입력 대비 모바일 '원터치 콤보'의 발동 속도 및 판정 비교 |
| **전투** | 피격/상태이상 판정 | 남마법사 특유의 '불사' 스킬이 던전에서 정상 작동하는지 확인 |
| **UI/UX** | 환경별 HUD 최적화 | PC(FHD/QHD)와 모바일(18:9 등) 해상도 전환 시 스킬 쿨타임 가독성 및 터치 영역 확인 |
| **성장** | 이벤트 보상 수령 | 점핑 이벤트 보상(강화권, 아바타 등)의 인벤토리 지급 및 실적용 여부 |
| **네트워크** | 입력 지연(Latency) | PC 버전의 키보드 입력과 모바일 터치 입력 간의 반응 속도(Input Lag) 체감 차이 측정 |

### 2.2 서브 TC (30~40%): 모바일 특화 및 환경 검증
| 구분 | 테스트 항목 | 상세 내용 (라이브 서버 기준) |
| :--- | :--- | :--- |
| **입력** | 외부 컨트롤러 대응 | 모바일 기기에 게임패드(8BitDo) 연결 시 남마법사 전용 스킬 매핑 정상 여부 |
| **환경** | 백그라운드 전환 | 플레이 중 전화 수신, 타 앱 전환 후 복귀 시 '남마법사' 리소스 로딩 오류 확인 |
| **성능** | 발열 및 배터리 | 30분 이상 연속 사냥 시 남마법사의 화려한 이펙트가 프레임(FPS)에 미치는 영향 |
| **네트워크** | 핸드오버 | Wi-Fi에서 LTE로 전환 시 남마법사의 스킬 위치 동기화 오류(고스트 현상) 확인 |

## 3. 테스트 입력물 및 환경 (Test Inputs & Environment)
* **테스트 환경:** 라이브 서버 (Mobile: S24/태블릿, PC: 공식 Native Client)
* **테스트 입력물:**
    * **계정 풀 (Account Pool):** Lv.80 점핑 캐릭터 계정
    * **캐릭터 구성:** 남마법사 전직 2종 (빙결사, 스위프트 마스터)
    * **외부 컨트롤러:** 8BitDo Gamepad

---

## 4. 라이브 비교 테스트 절차 (Execution Methodology)
1.  **동일 시나리오 반복:** PC와 모바일에서 동일한 던전(예: 빌마르크 제국 실험장)을 남마법사로 클리어하며 데이터 기록.
2.  **데이터 교차 검증:** PC에서 플레이 후 즉시 모바일로 접속하여 경험치, 재화, 스킬 세팅이 100% 동기화되는지 확인.
3.  **조작성 정성 평가:** PC의 '손맛'이 모바일의 '터치 최적화(슬라이드 스킬 등)'로 어떻게 치환되었는지에 대한 UX 분석 보고서 작성.

---

## 5. 리스크 분석 및 대응 (Risk Analysis)
> **[Risk 01] 네트워크 불안정**
> * **내용:** 라이브 서버 특성상 대규모 접속 시 발생하는 렉(Lag)이 남마법사의 빠른 이동기(스위프트 마스터)와 충돌하여 위치 렉 유발 가능성.
> * **대응:** 혼잡 채널과 원활 채널에서의 스킬 반응 속도 비교 데이터 확보.

> **[Risk 02] 기기 파편화**
> * **내용:** 안드로이드 OS 버전에 따라 남마법사 신규 이펙트가 검은색 박스로 출력되는 리소스 깨짐 위험.
> * **대응:** 주요 GPU 칩셋별(Adreno, Mali) 렌더링 검증 수행.

---

## 6. 테스트 산출물 (Outputs)
* **테스트 케이스 명세서:** 이슈별 구체적인 시나리오 및 검증 항목 목록.
* **테스트 실행 결과 보고서:** 실행 결과(Pass/Fail), 결함 추적 ID 및 심각도.
* **결함 보고서:** 상세 재현 단계, 스크린샷/로그가 포함된 JIRA 연동 문서.
* **플랫폼 비교 분석표:** PC vs Mobile 조작성 및 UI 가시성 비교 데이터.
