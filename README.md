# 🎮 DNF-M: 남마법사 - 빙결사 QA Project

본 프로젝트는 **던전앤파이터 모바일**의 신규 전직 '빙결사' 업데이트를 타겟으로 한 품질 보증(QA) 프로세스 포트폴리오입니다. 단순 기능 확인을 넘어, 최신 플래그십 기기(S24)와 멀티 플랫폼 환경에서의 기술적 리스크 분석 및 검증 전략을 담고 있습니다.

---

## 🚀 프로젝트 핵심 하이라이트

### 1. 리스크 기반 테스트 설계 (Risk-Based Testing)
* **메커니즘 분석:** 빙결사 고유의 '수속성 강제 부여', '강제 홀딩', '불사' 시스템의 로직 결함을 사전에 도출.
* **퍼포먼스 최적화:** S24(Snapdragon 8 Gen 3) 환경에서 고부하 파티클(각성기 '쥬데카') 시전 시 프레임 드랍 및 발열 제어 검증.
* **멀티 플랫폼 조작성:** 모바일 터치 조작과 블루투스 컨트롤러, PC 환경 간의 입력 지연 및 캔슬 로직 정합성 비교.

### 2. 플랫폼 특화 예외 상황 대응
* 네트워크 핸드오버(Wi-Fi ↔ 5G) 시 세션 유지력 검증.
* 저전력 모드(배터리 15% 이하) 하의 그래픽 최적화 렌더링 확인.

## 📅 프로젝트 관리 및 로드맵

### 📋 [QA 계획서 (QA Plan)](./QA_Plan.md)
> 프로젝트의 전체적인 범위, 일정, 리소스를 정의한 마스터 문서입니다.
* **테스트 범위:** 빙결사 신규 스킬셋, 시스템 연동, 플랫폼 호환성 검증.
* **일정 관리:** 분석(3일) → 설계(5일) → 실행 및 리그레션(7일)의 체계적 프로세스.
* **품질 목표:** P1(결정적) 결함 0건, P2(주요) 결함 수정률 100% 지향.
