# 🔍 신규 캐릭터 및 기능 분석 (Feature Analysis)

본 문서는 '남마법사' 업데이트의 핵심 기능을 분석하고, 발생 가능한 리스크를 사전에 파악하여 테스트 케이스의 근거를 마련하기 위해 작성되었습니다.

## 1. 캐릭터 메커니즘 분석

### 1.1 빙결사 (Glacial Master)
* **핵심 컨셉:** 수속성 근접 마법 딜러, 빙결 상태이상 특화.
* **주요 QA 포인트:**
    * 모든 스킬 시전 시 '수속성'이 강제 부여되는지 확인.
    * 빙결 상태이상 시 몬스터의 패턴이 정상적으로 멈추는지(홀딩 판정) 검증.
    * '불사' 패시브 발동 시 무색 큐브 조각 소모와 회복 메커니즘의 정합성.

### 1.2 스위프트 마스터 (Swift Master)
* **핵심 컨셉:** 바람을 이용한 기동성, 넓은 범위의 흡입(몰이) 기술.
* **주요 QA 포인트:**
    * '소닉 무브' 등 고속 이동 시 클라이언트와 서버 간의 위치 동기화(좌표 오차).
    * '폭풍의 눈' 등 흡입 스킬 사용 시 몬스터의 피격 박스가 중앙으로 정확히 모이는지 확인.
    * 공중 체류 상태에서 스킬 연계 시 애니메이션 끊김 현상 모니터링.

## 2. 하드웨어 및 환경 사양 분석 (Based on Official Spec)

제시된 권장/최소 사양을 바탕으로 기기별 집중 테스트 요소를 정의합니다.

| 구분 | 주요 분석 내용 | 테스트 집중 요소 |
| :--- | :--- | :--- |
| **권장 (S24)** | 최신 칩셋(Snapdragon 8 Gen 3) 및 Android 14 환경 | 고주사율 프레임 유지, 최상급 이펙트 렌더링 |
| **최소 (S10)** | 리소스 로딩 속도 및 메모리(4GB) 부족 가능성 | 앱 크래시(Crash), 저화질 이펙트 깨짐 여부 |
| **컨트롤러** | 블루투스 입력 신호와 게임 엔진 간의 반응성 | 입력 지연(Input Lag), 버튼 매핑 정상 동작 |

## 3. 리스크 도출
* **리소스:** 남마법사 전용 이펙트(한기, 바람) 추가로 인한 메모리 점유율 상승.
* **네트워크:** 기동성이 높은 캐릭터 특성상 네트워크 핑(Ping)에 따른 위치 렉 발생 확률 높음.
