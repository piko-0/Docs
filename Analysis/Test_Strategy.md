# [전략] 던파M 남마법사 업데이트 검증 전략

## 1. 테스트 접근 방식
- **리스크 기반 테스트 (RBT):** 신규 전직 2종의 핵심 스킬 및 이벤트 재화 수령 로직을 P1(최우선)으로 설정.
- **리그레션 테스트:** 남마법사와 파티 플레이 시 기존 클래스(버퍼 등)의 스킬 효과 적용 여부 검증.

### 2. 특수 테스트 설계 (Special Test Design)

실제 사용자 환경에서 발생할 수 있는 변수들을 통제하여, 남마법사 업데이트의 안정성 및 세션 유지력을 검증합니다.

#### 2.1. 사용자 환경 간섭 (Interrupt Test)
플레이 도중 발생하는 외부 신호 및 기기 상태 변화에 대한 대응력을 확인합니다.
* **전화/메시지 수신:** 보스전 또는 스킬 연계 중 전화 수신(팝업/전체화면) 시 앱이 백그라운드로 전환되어도 캐릭터의 스킬 위치나 판정 데이터가 꼬이지 않는지 검증합니다.
* **백그라운드 복귀:** 타 앱(카카오톡, 유튜브 등) 사용 후 재진입 시 남마법사의 신규 리소스(파티클 이펙트 등) 로딩 속도 및 서버 데이터 동기화 상태를 확인합니다.

#### 2.2. 네트워크 핸드오버 (Network Handover)
네트워크 환경이 급격히 변할 때의 세션 유지력과 데이터 무결성을 검증합니다.
* **네트워크 전환:** **Wi-Fi → LTE/5G**로 전환되는 시점에 '스위프트 마스터'의 초고속 이동기 사용 시 캐릭터 좌표가 맵 밖으로 튕기거나 이전 위치로 되돌아가는 롤백(Roll-back) 현상을 확인합니다.
* **비행기 모드/일시 차단:** 의도적인 네트워크 차단 후 재연결 시 '남마법사' 전용 전직 퀘스트 진행 데이터가 유실되지 않고 정상적으로 복구되는지 검증합니다.

#### 2.3. 이기종 기기 비교 검증 (Cross-Device Comparison)
서로 다른 하드웨어 스펙 및 폼팩터에서의 최적화와 사용자 경험(UX)을 비교 분석합니다.

| 검증 환경 | 주요 검증 항목 | 상세 내용 |
| :--- | :--- | :--- |
| **Galaxy S24 (Flagship)** | **발열 및 퍼포먼스** | 고해상도 설정에서 남마법사 스킬 이펙트 중첩 시 발열 스로틀링에 따른 FPS 변화 측정 |
| **Android Tablet** | **UI/UX 최적화** | 스마트폰 대비 넓은 화면비에서의 HUD 버튼(스킬 슬롯) 배치 및 터치 감도 분석 |
| **PC Client (Native)** | **입력 반응 및 안정성** | 키보드 입력 시 조작 반응 속도(Input Lag) 우위 확인 및 화면 전환(Alt+Tab) 시 안정성 비교 |

> **[Note]** 태블릿 기기 정보는 실기기 확인 후 업데이트 예정 (예: Galaxy Tab S10 Ultra 등)
